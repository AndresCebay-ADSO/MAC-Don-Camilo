01 Base del sistema
Homebrew
Obligatorio primero ⌨ Terminal
El gestor de paquetes de macOS. Sin esto no puedes instalar casi nada fácil desde terminal. Es literalmente el primer paso en cualquier Mac para desarrollo.
Terminal (Cmd + Espacio → "Terminal")
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Después de instalar, el terminal te pedirá que corras 2 comandos más para agregar Homebrew al PATH. Cópialos y ejecútalos — son específicos para tu usuario.
Oh My Zsh
Recomendado ⌨ Terminal
Mejora brutal del terminal. Muestra la branch de Git en la que estás, autocompletado, colores. Una vez lo usas no puedes vivir sin él.
Terminal
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
Git + Configuración
Obligatorio ⌨ Terminal
Ya tienes Git 2.49 en Windows. En Mac viene preinstalado pero la versión de Homebrew es más nueva y actualizable. Configura tu nombre y correo de GitHub.
Terminal
brew install git git config --global user.name "Tu Nombre" git config --global user.email "tu@correo.com" # Verificar: git --version

03 Stack PHP / Laravel
PHP 8.3 + Composer
Obligatorio ⌨ Terminal
Composer no lo tienes instalado actualmente — es el npm de PHP y es necesario para instalar Laravel y sus dependencias. También instalamos el Laravel Installer global para crear proyectos con un solo comando.
Terminal
# PHP 8.3 (compatible con Laravel 11): brew install php@8.3 echo 'export PATH="/opt/homebrew/opt/php@8.3/bin:$PATH"' >> ~/.zshrc echo 'export PATH="/opt/homebrew/opt/php@8.3/sbin:$PATH"' >> ~/.zshrc source ~/.zshrc # Composer (gestor de dependencias de PHP): brew install composer # Laravel Installer global: composer global require laravel/installer echo 'export PATH="$HOME/.composer/vendor/bin:$PATH"' >> ~/.zshrc source ~/.zshrc # Verificar todo: php -v composer --version laravel --version
Laravel Herd (reemplaza XAMPP)
Obligatorio ↓ Descarga directa
En Mac no se usa XAMPP. Herd es la herramienta oficial de Laravel para Mac: gestiona Nginx y dominios locales (.test) de forma nativa en Apple Silicon. Mucho más rápido que XAMPP. Arrastra tu proyecto a ~/Herd y accede como proyecto.test en el navegador.
↗ Descargar Laravel Herd (gratis)
Herd reemplaza XAMPP completamente. No instales XAMPP en Mac.
MySQL + TablePlus
Obligatorio ⌨ Terminal ↓ TablePlus: descarga
MySQL como motor de base de datos. TablePlus como cliente gráfico (reemplaza phpMyAdmin de tu XAMPP — mucho mejor en Mac). Nativo Apple Silicon, súper rápido.
Terminal
brew install mysql brew services start mysql # Primera configuración de seguridad: mysql_secure_installation
↗ Descargar TablePlus (gratis)
Si en el futuro usas Herd Pro (de pago), ya incluye MySQL integrado. En ese caso omite el brew install mysql para evitar conflictos de puerto 3306.
04 JavaScript / Frontend
Node.js + npm
Dev ⌨ Terminal
Ya tienes Node 22.15 en Windows. En Mac instala via nvm para poder cambiar versiones fácilmente. Necesario para compilar Tailwind CSS y Alpine.js con Vite en Laravel.
Terminal
# Instalar nvm vía Homebrew: brew install nvm mkdir -p ~/.nvm # Agregar al .zshrc: echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc echo '[ -s "$HOMEBREW_PREFIX/opt/nvm/nvm.sh" ] && \. "$HOMEBREW_PREFIX/opt/nvm/nvm.sh"' >> ~/.zshrc echo '[ -s "$HOMEBREW_PREFIX/opt/nvm/etc/bash_completion.d/nvm" ] && \. "$HOMEBREW_PREFIX/opt/nvm/etc/bash_completion.d/nvm"' >> ~/.zshrc source ~/.zshrc # Instalar Node LTS (versión estable recomendada): nvm install --lts nvm use --lts # Verificar: node -v npm -v

06 Herramientas finales
GitHub CLI + SSH Key
Dev ⌨ Terminal
El Mac nuevo no tiene ninguna clave SSH configurada — este paso es obligatorio para poder hacer push a GitHub sin escribir usuario y contraseña cada vez. El GitHub CLI facilita autenticarse y crear repos desde terminal.
Terminal
# Instalar GitHub CLI: brew install gh gh auth login # Generar SSH key para GitHub: ssh-keygen -t ed25519 -C "tu@correo.com" # Ver tu clave pública y copiarla: cat ~/.ssh/id_ed25519.pub # Agrégala en: # github.com → Settings → SSH and GPG Keys → New SSH Key
