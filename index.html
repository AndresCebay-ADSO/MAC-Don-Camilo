00 Instrucciones para el Setup
🖱️

Cómo copiar

Haga clic en el icono 📋 al lado de cada comando. El texto se guardará automáticamente en su memoria.

⌨️

Cómo pegar

En el Terminal, use Cmd + V para pegar y luego presione Enter para ejecutar cada paso.

📂

Instalar Apps

Cuando descargue una aplicación, recuerde arrastrar el icono a la carpeta de Aplicaciones de su Mac.

🚀

Abrir Terminal

Presione Cmd + Espacio, escriba "Terminal" y presione Enter para abrir la consola de comandos.

01 Base para el Equipo
Instalador Base (Homebrew)
PASO 1 ⌨ Terminal
Esta es la herramienta necesaria para que el Mac pueda cargar programas profesionales. Solo cópiela y ejecútela una vez.
Terminal
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Nota: Tras terminar, si el Terminal muestra instrucciones de "Next Steps", cópielas y ejecútelas también.
Perfil de Andrés (Git)
PASO 2 ⌨ Terminal
Establece el nombre y correo de Andrés en el sistema. Copie ambos pasos por separado.
Terminal
1️⃣ Paso 1: brew install git
2️⃣ Paso 2: git config --global user.name "Andrés Cebay"
3️⃣ Paso 3: git config --global user.email "cebayy044@gmail.com"

03 Stack PHP / Laravel
PHP 8.3 + Composer
Obligatorio ⌨ Terminal
Necesario para que el Mac entienda el código PHP de Andrés. Copie y ejecute estos pasos en orden.
Terminal
1️⃣ Paso 1: brew install php@8.3
2️⃣ Paso 2: echo 'export PATH="/opt/homebrew/opt/php@8.3/bin:$PATH"' >> ~/.zshrc
3️⃣ Paso 3: echo 'export PATH="/opt/homebrew/opt/php@8.3/sbin:$PATH"' >> ~/.zshrc
4️⃣ Paso 4: source ~/.zshrc
Herramientas Adicionales
5️⃣ Paso 5: brew install composer
6️⃣ Paso 6: composer global require laravel/installer
7️⃣ Paso 7: echo 'export PATH="$HOME/.composer/vendor/bin:$PATH"' >> ~/.zshrc
8️⃣ Paso 8: source ~/.zshrc
Verificar
php -v
composer --version
laravel --version
Laravel Herd (reemplaza XAMPP)
Obligatorio ↓ Descarga directa

↗ Descargar Laravel Herd (Gratis)

04 JavaScript / Frontend
Node.js + npm
Dev ⌨ Terminal
Herramientas para manejar el diseño y comportamiento de las webs. Siga los pasos numerados.
Terminal
1️⃣ Paso 1: brew install nvm
2️⃣ Paso 2: mkdir -p ~/.nvm
3️⃣ Paso 3: echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.zshrc
4️⃣ Paso 4: [ -s "$HOMEBREW_PREFIX/opt/nvm/nvm.sh" ] && \. "$HOMEBREW_PREFIX/opt/nvm/nvm.sh"
5️⃣ Paso 5: source ~/.zshrc
Configuración de Node
6️⃣ Paso 6: nvm install --lts
7️⃣ Paso 7: nvm use --lts

06 IA Local & Agentes
Ollama (Cerebro Local)
IA ⌨ Terminal
Esta es la herramienta principal para correr IAs en tu propio Mac sin depender de la nube. Al ser un M1 Max, funcionará muy rápido.
Terminal
brew install --cask ollama
Después de instalar, abre la aplicación "Ollama" desde tu carpeta de aplicaciones para iniciar el servidor.
Modelos IA Locales (Llama, Qwen, DeepSeek)
IA ⌨ Terminal
Descargue estos modelos recomendados para desarrollo. Se descargarán automáticamente la primera vez que los corra.
Terminal
1️⃣ Llama 3.1 (8B): ollama run llama3.1:8b
2️⃣ Qwen 2.5 Coder (32B): ollama run qwen2.5-coder:32b
3️⃣ DeepSeek Coder V2 (16B): ollama run deepseek-coder-v2:16b
¿Cómo correrlas después? Simplemente use el comando ollama run [nombre-modelo] en el terminal en cualquier momento.
Control de Agentes (Claude + Kimi)
IA ⌨ Terminal
Este comando especial permite conectar modelos en la nube como Kimi a través de la interfaz de Claude.
Terminal
ollama launch claude --model kimi-k2.5:cloud
